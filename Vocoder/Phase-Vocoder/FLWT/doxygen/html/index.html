Fast Lifting Wavelet Transform Module (FLWT) for Pitch Detection
Author
Terry Kong
Date
Mar. 9, 2015

Description

This is a fast implementation of the FLWT for pitch detection. The implementation works on integer data, which is the preferred data type on embedded systems. The algorithm is discussed in great detail in http://online.physics.uiuc.edu/courses/phys406/NSF_REU_Reports/2005_reu/Real-Time_Time-Domain_Pitch_Tracking_Using_Wavelets.pdf


Included in this class are an assortment of wrapper functions that help increase the consistency between pitch measurements. The most reliable of which involves using a median filter to smooth out the fluctuations in frequency.


This pitch detection algorithm works best when the audio has a singal strong fundamental harmonic. Audio with a mixture of fundamental harmonics, like a music track with many instruments, is not likely to be processed well. Also, The change in pitch with time needs to be relatively slow in order for the algorithm to estimatethe pitch reliably. If the algorithm decides the windowed audio is pitchless, it conservatively returns 0.
