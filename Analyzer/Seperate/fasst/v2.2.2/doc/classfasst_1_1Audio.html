Detailed Description

This class contains audio data. The audio data is stored in an Eigen::ArrayXXd object, which can be seen as an \(I \times N\)-array where \(N\) is the number of time frames and \(I\) is the number of audio channels (rows of the array are audio channels and columns of the array are audio samples.)
