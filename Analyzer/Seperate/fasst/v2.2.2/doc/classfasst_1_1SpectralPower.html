Detailed Description

This class represents either the excitation spectral power or the filter spectral power of a source. It has 4 NonNegMatrix objects to store W, U, G and H.

bool fasst::SpectralPower::equal ( const SpectralPower &  m,

This method is an equality test between the current SpectralPower object and the one passed in parameter. SpectralPower objects are equals if :

  • non negative matrix U are equals (according to a margin)
  • non negative matrix G are equals (according to a margin)
  • non negative matrix W are equals (according to a margin)
  • non negative matrix H are equals (according to a margin)
    Parameters
    m: SpectralPower object to be compared
    margin: relative margin tolerance expressed in %
    Returns
    true if SpectralPower are equals, false otherwise
