Detailed Description

This class represents a set of sources. In addition, it has an attribute for A which models the global mixing parameter containing each source. This parameter is stored as a VectorMatrixXcd which can be seen as a \(F\)-vector of \(I \times R\)-complex-matrices, where \(F\) is the number of frequency bins, \(I\) is the number of audio channels and \(R\) is the sum of the spatial covariance rank of each source. This class implements Wiener_Interface.

) const

This method is an equality test between the current Sources object and the one passed in parameter. Sources objects are equals if :

  • their number of frequency bins are equals
  • their number of time frames are equals
  • their number of channels are equals
  • the source number is equal
  • all sources composing the Sources object are equals (according to a margin)
    Parameters
    ss: Sources object to be compared
    margin: relative margin tolerance expressed in %
    Returns
    true if Sources are equals, false otherwise
