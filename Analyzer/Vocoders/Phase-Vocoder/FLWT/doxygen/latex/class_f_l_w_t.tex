Calculates the pitch of a set of data. 

====================================================

Uses the \hyperlink{class_f_l_w_t}{F\+L\+W\+T} described in Eric Larson and Ross Maddox\textquotesingle{}s paper. The algorithm was optimized for embedded systems that do not have dedicated hardware for floating point arithmetic by removing any floating point operation that was not deemed necessary to calculate the pitch. This of course decreases the accuracy of the algorithm, but at the same time increases the efficiency of the algorithm. If a more precise calculation of pitch is desired, check out the U\+R\+L below for a floating point implementation of this algorithm.
