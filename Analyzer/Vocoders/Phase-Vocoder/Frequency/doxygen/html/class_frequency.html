Returns the closest key frequency restricted to a scale.

====================================================

Returns the key number restricted to a scale: if bigger than max or smaller than min, returns NUM_OF_KEYS or 0 respectively. The key that specifies the beginning of the scale can be anywhere between 1 and 88, but the frequency returned can be anywhere between the highest and lowest frequency piano key frequency.
