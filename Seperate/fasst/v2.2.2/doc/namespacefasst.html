Detailed Description

This class represents an base class for time-frequency representations. It provides a default constructor to construct private members

This class allows the computation of the time-frequency representation of some audio signal using the ERBLET representation. This class derives Abstract_TFRepr.

This class is a minimal interface for Wiener Filtering purpose
