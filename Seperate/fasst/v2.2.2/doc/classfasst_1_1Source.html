Detailed Description

This class represents a source with all of its 9 parameters: one MixingParameter and eight NonNegMatrix. In addition, it has an attribute for V which models the entire spectral power of the source and is stored in a Eigen::MatrixXd object which can be seen as a \(F \times N\)-matrix, and an attribute for R which models the spatial covariance matrix and is stored in a VectorMatrixXcd object which can be seen as a \(F\)-vector of \(I \times I\)-complex-matrices.
